no linux:
cd para dentro do diretorio do trabalho
make -> compila as classes
make write -> escreve os objetos para serem adicionados no jogo
make run -> roda o jogo
no windows:
clicar em compilador.bat-> compila as classes
clicar em escreve.bat-> escreve os objetos para serem adicionados no jogo
clicar em executa-> roda o jogo
